# breakdown goal

## Purpose: 

To break down a daunting goal into actionable steps.

## Prompt:

 I want to write a book. Think step-by-step to create a 30-day action plan for the first draft.

# Example Input: 

[No additional input needed]

# Example Output:

1.Days 1-3: Define the premise and create a 10-chapter outline.
2.Days 4-7: Character sketches and world-building.
3.Days 8-28: Write 2,000 words per day (focus on quantity over quality).
4.Days 29-30: Review the draft and note major plot holes for the second pass.
