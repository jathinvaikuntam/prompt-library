# categorize diet

## Purpose: 

To categorize tasks using the Eisenhower Matrix.

## Prompt: 

Categorize tasks as: Urgent/Important, Not Urgent/Important, Urgent/Not Important, or Neither.

# Example:

 "Fixing a leaking pipe" -> Urgent/Important

# Example Input: 

"Planning a vacation for next year"

# Example Output: 

Not Urgent/Important
