# high level coaching

## Purpose: 

To receive high-level time management coaching.

## Prompt: 

You are a world-class Executive Assistant. Suggest a system for managing an overflowing inbox for a busy CEO.

# Example Input:

 "I have 500 unread emails and 50 new ones every hour."

# Example Output: 

"We will implement the 'OHIO' method (Only Handle It Once). I will set up filters to divert newsletters to a 'Read Later' folder. For the remaining, we will use three folders: @Action (requires a task), @Waiting (pending others), and @Archive. I will clear everything else, and you will only check the @Action folder twice daily."
