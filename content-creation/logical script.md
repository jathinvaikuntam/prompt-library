# logical script 

## Purpose
 To ensure a script is logically structured.

## tags
chain of thought

## Prompt 
```
I want to create a 60-second TikTok about productivity. Think step-by-step: first identify the hook, then the three main tips, and finally the call to action.

topic : {{topic_name}}
```

# Example Input 

Topic: Beating Procrastination.

# Example Output:

1.Hook: "Stop scrolling! You're procrastinating because the task feels too big, not because you're lazy."
2.Tip 1: The 5-second rule (count down and move).
3.Tip 2: The 2-minute rule (if it takes 2 mins, do it now).
4.Tip 3: Eat the frog (do the hardest thing first).
5.DO IT' if you're going to finish one task today!.
