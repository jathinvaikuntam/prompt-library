# extract specific data

## Purpose: 

To extract specific data points from unstructured text.

## Prompt: 

Extract the Company Name and Revenue from the text.

## Example: 

"Global Corp made 5 million" -> {"Company": "Global Corp", "Revenue": "5M"}

# Example Input: 

"TechStart Inc. reported earnings of twelve billion dollars this year."

# Example Output: 
{"Company": "TechStart Inc.", "Revenue": "12B"}
